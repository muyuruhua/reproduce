## 审核员操作流程

### 第 1 步：打开 Excel 找到漏洞

打开 `vulnerability/bftpd_漏洞发现_20260606145639.xlsx`，Vulnerability List sheet 中看到：

| # | Severity | Category | CWE | Description | CVE Pattern |
|---|----------|----------|-----|-------------|-------------|
| 1 | 4 (HIGH) | **0x0020** | CWE-93 | CRLF injection in FTP command arguments | CVE-2026-39983 |
| 2 | 4 (HIGH) | **0x0004** | CWE-696 | RNTO without prior RNFR accepted by server | N/A |
| 3 | 5 (CRITICAL) | **0x0008** | CWE-200 | Sensitive file content leaked in response | CVE-2024-42650 |

### 第 2 步：Category 值 → 种子文件名

```bash
ls bftpd/seeds/violations/
# viol_1_id_000005_sev_4_cat_0020  ← cat_0020 = 0x0020 匹配Excel第1行
# viol_2_id_000006_sev_4_cat_0024  ← cat_0024 = 0x0024 复合类型
# viol_3_id_000007_sev_5_cat_0008  ← cat_0008 = 0x0008 匹配Excel第3行
```

**种子文件名中的 `cat_0020` 就是 Excel Category 列的值 `0x0020`。**

### 第 3 步：一条命令复现

```bash
bash scripts/replay_logical_vuln.sh bftpd \
    bftpd/seeds/violations/viol_1_id_000005_sev_4_cat_0020
```

### 第 4 步：看结果

```
Verifying FTP ... req=122B resp=423B
  [HIGH] CRLF_INJECTION: CRLF injection in FTP command: b'...\r\nSY?T\r\n...'
  [HIGH] AUTH_BYPASS: Unauthenticated b'MKD todeletenow' (code 200)
[CONFIRMED] 2 oracle violation(s) verified    ← 漏洞已独立复现
```

## 全部 10 个目标：Excel Category → 种子 → 命令

审核员看到 Excel 中某个 Category 值，直接在下面找到对应的种子文件，执行命令即可。

### Category 掩码速查

| Category | 名称 | 一句话解释 |
|:---:|------|------|
| 0x0001 | AUTH_BYPASS | 没登录就能干特权操作 |
| 0x0004 | STATE_VIOLATION | 跳过RFC规定的协议状态转换 |
| 0x0008 | INFO_LEAK | 响应中泄露了密码文件/系统信息 |
| 0x0010 | PATH_TRAVERSAL | `../` 越权访问文件系统 |
| 0x0020 | INJECTION | 命令参数含嵌入换行符(CRLF)，可走私命令 |
| 0x0040 | DOS | 畸形输入导致服务不可用 |
| 0x0080 | RESOURCE_EXHAUST | 畸形输入消耗完内存/连接/句柄 |
| 0x0100 | ISOLATION | FTP Bounce(PORT指向内网IP被接受) |
| 0x0400 | SMUGGLING | HTTP请求走私(多CL / CL+TE共存) |

## 快速开始

```bash
# 1. 导入镜像（首次）
for f in docker_images/*.tar.gz; do docker load -i "$f"; done

# 2. 一键演示
bash quickstart.sh --demo

# 3. 手动复现指定漏洞（Excel中看到 Category=0x0020）
bash scripts/replay_logical_vuln.sh bftpd bftpd/seeds/violations/viol_1_id_000005_sev_4_cat_0020
```
